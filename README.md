

# **StudentConnect**  
**Find shared spaces, explore student deals, and discover what's around you — all in one place.**

---

## **Project Overview**  
**Project Name**: StudentConnect  
**Developer**: Jaini Shah  
**Supervisor/Professor**: Ray Toal  
**Date**: December 9, 2024  
**Version**: 1.0  

**StudentConnect** is an innovative web application designed to simplify student life by connecting them through **apartment listings** and a **marketplace** for sharing items. The platform fosters collaboration, sustainability, and community engagement, offering students a streamlined way to:  
- Find housing solutions.  
- Share resources with peers.  
- Thrive academically and personally.  

---

## **Core Features**  
1. **Student Dashboard**:  
   - Personalized dashboards for managing activities and resources.

2. **Apartment Listings**:  
   - Share and discover apartment listings.  
   - Filter based on school, budget, and preferences.  
   - Connect with peers for shared housing opportunities.

3. **Marketplace**:  
   - Share or sell items like books and electronics.  
   - Browse listings using advanced filters.  
   - Promote sustainable reuse within the student community.  

---

## **Mission and Vision**  
- **Mission**:  
  Empower students by connecting them through apartment listings and a marketplace for sharing items, fostering collaboration and sustainability.

- **Vision**:  
  A future where students effortlessly find housing, share resources, and build meaningful connections to thrive academically and personally.

## **Technologies Used**  
- **Frontend**: React.js  
- **Backend**: Node.js with Express.js  
- **Database**: Supabase (PostgreSQL)  
- **Authentication**: JWT (JSON Web Token)  
- **Deployment**: Netlify (Frontend) and Heroku (Backend)  

---

## **How to Get Started**  
1. Clone the repository:  
```bash
git clone https://github.com/DevJaini/StudentConnect.git
```
2. Install dependencies for both frontend and backend:  

Frontend:
  ```bash
   cd client
   npm install
  ```

Backend:
  ```bash
   cd api
   npm install
   ```
3. Set up the database using Supabase or your preferred PostgreSQL instance.  

4. Add your environment variables:  
   - **Backend**: Add a `.env` file with your JWT secret and database credentials.  
   - **Frontend**: Add API base URL in the `.env` file.  

5. Start the development servers:  


Frontend:
  ```bash
   cd client
   npm start
  ```

Backend:
  ```bash
   cd api
   node server.js
   ```
